Каждый элемент входного файла имеет следующую структуру:
шифр_дисциплины,название_дисциплины,тип_дисциплины,кафедра,количество_часов_лекци
и, количество_часов_практических, количество_часов_консультаций;
Например: ИТУ-2, Дискретная математика, обязательная_профессиональная, ИУС, 30, 24, 10;
При этом шифр_дисциплины – состоит из прописных букв количеством от 3 до 5,
затем тире и номер семестра, в котором изучается дисциплина, число от 1 до 8 (ИТУ-6
дисциплина изучается в 6 семестре на специальности ИТУ), тип_дисциплины -
обязательно_профессиональная, обязательно_общая, выборочно_общая,
выборочно_профессиональная (в структуре использовать тип enum), название_дисциплины - может
содержать несколько слов, разделенных пробелами, максимальная длина – 30 символов; кафедра
- от двух до пяти прописных букв (абревиатура типа ИУС, СТ, ШИ), количество_часов_лекций,
количество_часов_практических, количество_часов_консультаций - числа в диапазоне от 0 до
50, обязательно парные. Данные по дисциплинам обособляются знаком «;». Необходимо
выводить следующую информацию:
   а) информацию о дисциплинах, которые преподаются заданной кафедрой;
   б) информацию о дисциплине, общее количество часов преподавания для которой есть
максимальной;
   в) вывести для каждой кафедры количество преподаваемых дисциплин и их список.
